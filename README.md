Contact Management System

A simple Contact Management System built using HTML, CSS, and JavaScript.
This project allows users to add, edit, delete, and store contact details using the browser’s Local Storage.

Features

Add new contacts (Name, Phone Number, Email)

View saved contacts

Edit existing contacts

Delete contacts

Data persistence using Local Storage

Responsive and user-friendly interface

Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

Local Storage

Project Structure

How to Run the Project

Clone the repository:

git clone https://github.com/your-username/contact-management-system.git


Open the project folder.

Open index.html in any modern web browser.

How It Works

Contact data is stored as JavaScript objects.

All data is saved in the browser’s Local Storage.

Contacts remain available even after refreshing the page.

Edit functionality allows updating existing contact details.
